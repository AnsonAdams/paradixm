Summary of History of Control Alt Delete

David Bradley helped build IBM’s new personal computer. Yet, when the computer encountered a glitch, then they had to manually restart the system. So Bradley created a keyboard shortcut ctrl + alt + del that triggered a system reset.

The reason why those three keys were chosen was due to their placement since it wasn’t likely for all three keys to be pressed at the same time on accident.

Although Bradley had created the shortcut to save time and help his fellow programmers, it was never intended for the customers. However, it wasn’t until much later on that Bradley’s shortcut became well-known.
