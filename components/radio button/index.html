Radio Button

A type of button, typically presented with other radio buttons to allow users to decide exclusively on one choice from a set of related and predetermined options.

Typical Appearance

Radio buttons are usually circular in appearance and are accompanied to the right with a text label describing the given option. When inactive, the circles are not colored in, giving it an "empty" look to suggest that it is not currently active. When active, a dot concentric to the circle or an entirely different color to the background "fills" the circle, indicating that it is currently chosen.
