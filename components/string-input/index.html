--- title: String Input layout: component ---
Every computer nowadays requires some form of text input from the user. This input is often accepted with a string input field.

The string input field is used when the program requires the user to enter text information.

Typical Appearance


The most basic form of a string input field is a one-line rectangular box.

For inputs that require more than one-line, a larger string input field can be used.
